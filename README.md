1️⃣ What is the difference between var, let, and const?
  
   var: It is the old way of declaring variables.
   let: Its value can be changed later.
   const: Once a value is set, it cannot be changed.
    
2️⃣ What is the spread operator (...)?

   Spread operator (...): It is used to expand the elements of an array or object into individual elements.

3️⃣ What is the difference between map(), filter(), and forEach()?

   map(): It creates a new array by applying a function to each element.
   filter(): It creates a new array with only the elements that meet a specific condition.
forEach(): It performs a specific action for each element of an array, but it does not return a new array.

4️⃣ What is an arrow function?
       
   Arrow function is a shorter and modern way to write functions. It does not require the function keyword and makes the code cleaner.
   Regular Function:
          function add(a, b) {
            return a + b;
          };
      
  Arrow Function:
         const add = (a, b) => a + b;

5️⃣ What are template literals?

 Template Literals is an advanced way to write strings, where the backtick ( ` ) symbol is used. Inside it, variables can be used directly.
